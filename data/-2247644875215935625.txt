						第二届“爱国奋斗”精神研讨会在创新港举行 	
					</div>
<div class="bmnr_con_yinti"> </div>
<div class="bmnr_con_biaoti"><div id="div1" style="font-size:12px; class:#ccc; font-weight:100;height:20px"></div></div>
<div class="bmnr_con_con">
<div id="zoom"><a class="thumbnail" href="#"><img align="center" height="262" src="4c87e59e2d59a8983a8eaab49a70415f_img_201_254_389_262.jpg" width="389"/><span><img src="4c87e59e2d59a8983a8eaab49a70415f_img_201_254_389_262.jpg" width="600"/></span></a><br/>　　<strong>本报讯</strong> 12月11日上午，由光明日报社、陕西省委宣传部和我校联合举办的第二届“爱国奋斗”精神研讨会在中国西部科技创新港涵英楼一楼E105教室举行。来自北京大学、清华大学、上海交通大学等30余所兄弟高校领导及专家代表，中陕媒体记者、学校相关单位负责人及师生代表200余人齐聚一堂，深入学习研讨教育系统广大知识分子的感人故事和奉献报国精神。 <br/>　　在习近平总书记对我校15位西迁老教授来信作出重要指示两周年之际，在中国西部科技创新港首次举行“爱国奋斗”精神研讨会，深入挖掘其思想内涵和时代价值，对于广大知识分子乃至全社会更好弘扬爱国奋斗精神、建功立业新时代具有重要示范意义。 <br/>　　光明日报社副总编辑李春林，中央马克思主义理论研究与建设工程首席专家、武汉大学教授骆郁廷，陕西省委宣传部副部长单红，陕西省委教育工委副巡视员梁爱芳，我校党委书记张迈曾，党委常委、统战部部长张定红等出席会议。学校党委常委、宣传部部长成进主持会议。 <br/>　　<strong>爱国奋斗，就是西迁精神在今天闪现的时代价值 </strong><br/>　　张迈曾在致辞中指出，习近平总书记对弘扬西迁精神作出重要指示，就是希望广大知识分子高扬爱国奋斗旗帜，以先进的思想文化引领社会。今天学习弘扬西迁精神，要赋予其新的时代内涵，即知识分子应更坚定、清晰地肩负起自己的政治使命；必须持续不断地艰苦奋斗，在创造中收获快乐；要更多强调集体，共同承担伟大战略目标的实现。 <br/>　　单红指出，两年来，陕西省委省政府深入贯彻落实习近平总书记重要指示精神，认真开展学习传承践行西迁精神，争当西迁精神新传人在三秦大地蔚然成风。省委宣传部按照中央和省委的要求，精心组织，广泛宣传，在全省各地和各行业掀起了学习西迁精神的热潮。习近平总书记的重要指示已经成为激励全国广大知识分子和全省教育科技工作者以及西安交大师生爱国奋斗、开拓创新的强大动力。 <br/>　　李春林分享了对认识和弘扬西迁精神的深入理解。他认为，第一，要在百年未有之大变局中深入认识和大力弘扬西迁精神。第二，要在开展“不忘初心、牢记使命”主题教育的大背景下深入认识和大力弘扬西迁精神。第三，要在充分发挥知识分子重要作用的过程中认识和弘扬西迁精神。西安交大用不到1000天的时间建成创新港，背后的精神支撑就是西迁精神。服务知识分子、团结知识分子一直以来是光明日报的初心使命，这一点和西安交大有着共同的思想认识和价值追求。光明日报社一定会以高度的政治责任感、饱满的政治热情，和西安交大把西迁精神研讨和弘扬活动一直做下去，越办越好。 <br/>　　<strong>爱国奋斗，就是要在实际行动中践行西迁精神</strong> <br/>　　为了形成涵盖广、质量高的会议成果，光明日报社、陕西省委宣传部和我校在会前联合开展了以“爱国不忘初心、奋斗牢记使命”为主题的论文征集活动，共收到投稿论文65篇，共有31篇论文获奖，其中一等奖5篇，二等奖12篇，三等奖14篇。光明日报社教育部主任田延辉宣读优秀论文评选结果，骆郁廷、张定红为一等奖获奖者颁奖。 <br/>　　会上，张迈曾、李春林、梁爱芳共同启动高教系统“最美爱国奋斗人”微视频征集活动。该活动由教育部指导，光明日报社、陕西省委宣传部和我校共同主办，我校新闻与新媒体学院承办，活动时间从2019年12月一直持续至2020年10月，参与者可在我校新闻与新媒体学院网站主页下载报名表并填写相关信息。 <br/>　　本次研讨会还特别邀请了骆郁廷教授作主题为“新时代爱国主义教育的破与立”的报告。他从破历史虚无主义，立历史唯物主义；破西方舆论诋毁，立中国国家形象；破崇洋媚外现象，立深厚爱国情怀；破精致利己主义，立使命担当精神四个方面，深入解读了新时代爱国主义教育破与立的命题。 <br/>　　当天下午，还同步举行了弘扬爱国奋斗精神案例交流、爱国奋斗精神获奖论文交流、时代巾帼弘扬爱国奋斗精神、当代青年弘扬爱国奋斗精神、统一战线弘扬爱国奋斗精神、学生工作主题六个分研讨会，围绕主题进一步深入研讨，让爱国奋斗精神往实里走、往深里走、往心里走，根植在每个人的内心深处。 <br/>　　<strong>（文/胡晓楠 图/郭小龙）</strong></div>
